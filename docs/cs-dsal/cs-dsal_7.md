# 第七章：总结

在阅读这本书的许多页面时，你已经学到了很多关于各种数据结构和算法的知识，这些知识可以在开发 C#语言应用程序时使用。数组、列表、栈、队列、字典、哈希集、树、堆和图，以及相应的算法——这是相当广泛的主题，不是吗？现在是时候总结这些知识了，同时提醒你一些特定结构的特定应用。
