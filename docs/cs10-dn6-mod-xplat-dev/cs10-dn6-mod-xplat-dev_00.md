# 前言

有一些编程书籍长达数千页，旨在成为 C#语言、.NET 库、网站、服务和桌面以及移动应用程序等应用模型的全面参考。

这本书与众不同。它简洁而旨在成为一个轻快、有趣的阅读，充满了每个主题的实用实践指南。全面的叙述广度是以一些深度为代价的，但如果你愿意，你会发现许多探索更多内容的指引。

这本书同时是一个逐步学习现代 C#实践的跨平台.NET 的指南，也是对可以用它们构建的主要类型的实际应用程序的简要介绍。这本书最适合初学者和有经验的 C#和.NET 程序员，或者过去曾使用 C#但感觉被过去几年的变化抛在了后面的程序员。

如果你已经有使用旧版本 C#语言的经验，那么在*第二章* *说 C#*的第一部分，你可以查看新语言特性的表格并直接跳转到它们。

如果你已经有使用旧版本的.NET 库的经验，那么在*第七章* *打包和分发.NET 类型*的第一部分，你可以查看新库特性的表格并直接跳转到它们。

我会指出 C#和.NET 的一些酷炫之处和陷阱，这样你就可以快速给同事留下深刻印象并提高工作效率。我不会为了解释每一个小事情而减慢速度并让一些读者感到无聊，我会假设你足够聪明，可以通过谷歌来解释一些相关但不必在印刷书中包含的主题。

# 本书涵盖内容

*第一章* *你好，C#! 欢迎，.NET!*，是关于设置你的开发环境，并使用 Visual Studio 或 Visual Studio Code 来创建最简单的 C#和.NET 应用程序。对于简化的控制台应用程序，你将看到 C# 9 中引入的顶级程序特性的使用。为了学习如何编写简单的语言结构和库特性，你将看到.NET 交互式笔记本的使用。你还将了解一些寻求帮助的好地方，以及联系我获取帮助或给我反馈以改进这本书和未来版本的方式，通过它的 GitHub 存储库。

*第二章* *说 C#*，介绍了 C#的版本，并列出了引入新功能的版本。我解释了你每天将用来编写应用程序源代码的语法和词汇。特别是，你将学习如何声明和处理不同类型的变量。

*第三章* *控制流、类型转换和异常处理*，涵盖了使用运算符对变量执行简单操作，包括比较，编写做出决策的代码，C# 7 到 C# 10 中的模式匹配，重复一组语句，并在不可避免地发生异常时编写防御性代码来处理异常。

第四章，编写、调试和测试函数，是关于遵循“不要重复自己”（DRY）原则，通过使用命令式和函数式实现风格编写可重用函数。您还将学习如何使用调试工具来追踪和消除错误，监视代码执行过程中诊断问题，并严格测试代码以消除错误并确保在部署到生产环境之前的稳定性和可靠性。

第五章，使用面向对象编程构建自己的类型，讨论了类型可以拥有的各种成员类别，包括用于存储数据的字段和用于执行操作的方法。您将使用面向对象编程（OOP）概念，如聚合和封装。您将了解元组语法支持和`out`变量、默认文字和推断元组名称，以及如何使用`record`关键字、`init`-only 属性和 C# 9 中引入的`with`表达式定义和处理不可变类型。

第六章，实现接口和继承类，解释了使用 OOP 从现有类型派生新类型。您将学习如何定义运算符和局部函数，委托和事件，如何实现关于基类和派生类的接口，如何覆盖类型的成员，如何使用多态，如何创建扩展方法，如何在继承层次结构中的类之间进行转换，以及 C# 8 中引入的可空引用类型的重大变化。

第七章，打包和分发.NET 类型，介绍了.NET 的版本，并展示了哪些版本引入了新的库功能，然后介绍了符合.NET 标准的.NET 类型以及它们与 C#的关系。您将学习如何在支持的操作系统上编写和编译代码：Windows、macOS 和 Linux 变体。您将学习如何打包、部署和分发自己的应用程序和库。

第八章，使用常见的.NET 类型，讨论了允许您的代码执行常见实际任务的类型，如操作数字和文本，日期和时间，将项目存储在集合中，处理网络和操作图像，以及实现国际化。

第九章，与文件、流和序列化一起工作，涵盖了与文件系统的交互，读写文件和流，文本编码，以及 JSON 和 XML 等序列化格式，包括`System.Text.Json`类的改进功能和性能。

第十章，使用 Entity Framework Core 处理数据，解释了如何读写关系数据库，如 Microsoft SQL Server 和 SQLite，使用名为 Entity Framework Core（EF Core）的对象关系映射（ORM）技术。您将学习如何定义映射到数据库中现有表的实体模型，以及如何定义可以在运行时创建表和数据库的 Code First 模型。

第十一章，使用 LINQ 查询和操作数据，教授您关于语言集成查询（LINQ）的知识，这是一种语言扩展，可以处理项目序列并对其进行过滤、排序和投影到不同的输出。您将了解并行 LINQ（PLINQ）和 LINQ 到 XML 的特殊功能。

第十二章，使用多任务改进性能和可伸缩性，讨论了允许多个操作同时发生以提高性能、可伸缩性和用户生产力。您将了解`async Main`功能以及如何使用`System.Diagnostics`命名空间中的类型来监视代码以测量性能和效率。

第十三章《介绍 C#和.NET 的实际应用》向您介绍了可以使用 C#和.NET 构建的跨平台应用程序类型。您还将构建一个 EF Core 模型来表示将在本书的其余章节中使用的 Northwind 数据库。

第十四章《使用 ASP.NET Core Razor Pages 构建网站》是关于学习使用 ASP.NET Core 在服务器端构建现代 HTTP 架构网站的基础知识。您将学习如何实现被称为 Razor Pages 的 ASP.NET Core 特性，它简化了为小型网站创建动态网页，并了解构建 HTTP 请求和响应管道。

第十五章《使用模型-视图-控制器模式构建网站》是关于学习如何以易于单元测试和团队程序员管理的方式构建大型复杂网站的。您将了解启动配置、身份验证、路由、模型、视图和控制器。

第十六章《构建和使用 Web 服务》解释了使用 ASP.NET Core Web API 构建后端 REST 架构 Web 服务，以及如何使用工厂实例化的 HTTP 客户端正确消耗它们。

第十七章《使用 Blazor 构建用户界面》介绍了如何使用 Blazor 构建可以在服务器端或客户端 Web 浏览器内执行的 Web 用户界面组件。您将了解 Blazor 服务器和 Blazor WebAssembly 之间的区别，以及如何构建更容易在两种托管模型之间切换的组件。

第十八章《构建和使用专业服务》向您介绍了使用 gRPC 构建服务，使用 SignalR 在服务器和客户端之间实现实时通信，使用 OData 公开 EF Core 模型，以及使用 Azure Functions 在云中响应触发器的主机功能。

第十九章《使用.NET MAUI 构建移动和桌面应用程序》向您介绍了如何为 Android、iOS、macOS 和 Windows 构建跨平台移动和桌面应用程序。您将学习 XAML 的基础知识，该知识可用于定义图形应用程序的用户界面。

第二十章《保护您的数据和应用程序》是关于使用加密保护数据免受恶意用户查看，以及使用哈希和签名防止数据被篡改或损坏。您还将了解身份验证和授权，以保护应用程序免受未经授权的用户。

附录《测试您的知识问题的答案》包含了每章末尾的测试问题的答案。
